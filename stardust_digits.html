Welcome to Stardust Digits! This tool will help you convert numbers between hexadecimal, octal, binary, and decimal number systems. Hexadecimal is a 16-based number system, it uses the digits 0-9 and the letters A-F. Octal is a 8-based number system, it uses digits 0-7. Binary is a 2-based number system(numbers 0 and 1). Decimal is the most common number system. It uses symbols from 0 to 9.
